# Product Brief

## Problem

- Codex Lattice helps coding agents keep implementation, verification, and durable docs aligned.

## Users

- Developers using Codex with reusable skills, hooks, custom agents, MCP search, and docs-first workflows.

## Scope

- Installable Codex harness configuration and operating rules.

## Non-Goals

- Replacing Codex built-in `/goal`, `/plan`, `/review`, `/diff`, `/agent`, `/mcp`, or `/plugins`.
- Automatically mutating code from hooks.

## Open Questions Before PRD

- Which docs should be mandatory for each repository type can be tuned per team after adoption.
